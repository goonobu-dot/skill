# Vibe Deck for D200X Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a Codex Micro–like live agent status deck on Ulanzi D200X for Claude, Codex, and Cursor, with Japanese manuals.

**Architecture:** A local Node bridge normalizes agent state from each desktop tool; an Ulanzi Studio JS plugin paints multi-state colored agent keys via `setState`; three ProfilesV2 presets map Micro-parity commands plus D200X extras.

**Tech Stack:** Node.js 20+, TypeScript (bridge), Ulanzi Studio JavaScript plugin (`Type: JavaScript`, websocket host/port from `argv`), PNG status tiles, JSON ProfilesV2 presets, macOS LaunchAgent optional installer.

2. `com.vibe.deck.status.focus` — focus agent in slot

Protocol (from Ulanzi Discord plugin):

```js

send(uuid, 'state', { param: { statelist: [{ key, actionid, type: 0, state: index }] } })
```
