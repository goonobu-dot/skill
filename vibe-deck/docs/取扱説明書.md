# Vibe Deck for D200X 取扱説明書

OpenAI Codex Micro に近い「エージェントの色監視＋手元操作」を、Ulanzi D200X 上で Claude / Codex / Cursor 向けに実現するセットです。

## 1. 同梱物

| 項目 | 内容 |
|------|------|
| Bridge | エージェント状態を集約する常駐プロセス |
| Plugin | 色タイルをライブ更新する Ulanzi プラグイン |
| Presets | `Vibe · Claude` / `Vibe · Cursor`（Codex は公式 `Codex_D200X` を利用） |
| 本説明書 | セットアップと使い方 |
| クイックリファレンス | キー配置の一覧 |

## 2. 必要なもの

- Ulanzi D200X（USB 接続）
- Ulanzi Studio 3.1+（Apple Silicon）
- macOS 12+
- 対象アプリ: Claude.app / ChatGPT.app（Codex）/ Cursor.app
- Node.js 20+

## 3. セットアップ（約10–20分）

```bash
cd /path/to/skill/vibe-deck
./scripts/install.sh
```

その後:

1. **Ulanzi Studio を再起動**
2. プロファイル一覧に `Vibe · Claude` などが増えていることを確認
3. **システム設定 → プライバシーとセキュリティ → アクセシビリティ** で Ulanzi Studio を許可
4. ChatGPT / Claude / Cursor にサインインしておく

デモで色だけ確認したい場合:

```bash
pkill -f vibe-deck/bridge/dist/index.js || true
VIBE_DECK_DEMO=1 node bridge/dist/index.js
```

## 4. 色とアイコンの意味（Codex Micro 互換）

ステータスキーは **色＋記号＋日本語ラベル** で状態を示します（色だけでも判別できます）。

| 見た目 | 状態 | ラベル | 意味 |
|--------|------|--------|------|
| 白＋一時停止 | idle | 待機中 | 待機 |
| 青＋軌道 | thinking | 考え中 | 思考中／作業中 |
| 緑＋チェック | done | 完了 | 完了 |
| 橙＋吹き出し? | needs_input | 要確認 | 承認・入力待ち |
| 赤＋警告 | error | エラー | エラー |
| 灰＋空き丸 | empty | 空き | 未使用スロット |

橙・赤は優先表示され、上のスロットに寄ります。

アイコンを再生成する場合:

```bash
python3 scripts/generate-icons.py
# その後 Ulanzi Studio を再起動
```

## 5. キーマップ（各プリセット共通レイアウト）

### ページ1: Control

| 位置 | 機能 |
|------|------|
| 上段（5キー） | Agent Slot 1–5（色＋アイコン＋日本語ラベル） |
| Accept / Reject | 提案の承認／拒否 |
| New / Stop | 新規セッション／停止 |
| Claude / Codex / Cursor | 各デスクトップアプリを前面化 |
| Mic | 音声入力（アプリの Dictation ショートカット） |

### ダイヤル

| Dial | 機能 |
|------|------|
| 1 | Effort / モード相当のショートカット |
| 2 | ステータス再取得（Refresh） |
| 3 | チャットスクロール |

### ページ2: Prompt Pack

Design / Implement / Review / Fix / Test / Explain / Commit / Summary の定型プロンプトを挿入。

## 6. ブリッジについて

- 既定 URL: `http://127.0.0.1:17823`
- `GET /health` … 生存確認
- `GET /status?tool=codex|claude|cursor` … スロット状態 JSON
- ライブ色にはブリッジ起動が必要です（止まっていてもショートカットキーは使えます）

### ツール別の精度

| ツール | 精度 | 備考 |
|--------|------|------|
| Codex | 高〜中 | `~/.codex` や ChatGPT 連携シグナルを利用 |
| Claude | 中 | `claude agents --json` / daemon を利用 |
| Cursor | 低〜中 | 公開ローカルAPIが弱いためベストエフォート |

## 7. Codex Micro との差分

| Micro | Vibe Deck |
|-------|-----------|
| RGB メカニカルキー | LCD 色タイル |
| 6 Agent Keys | 8 スロット |
| 専用ジョイスティック | Prompt Pack＋ハブキーで代替 |
| ChatGPT 専用 | Claude / Codex / Cursor の3プリセット |
| 本体エッジ発光 | メイン画面／キー色で表現 |

## 8. トラブルシュート

**色が変わらない**

1. `curl http://127.0.0.1:17823/health`
2. Ulanzi Studio 再起動
3. プラグインが `~/Library/Application Support/Ulanzi/UlanziDeck/Plugins/com.vibe.deck.status.ulanziPlugin` にあるか確認
4. デモモードでタイルが回るか確認

**キーが効かない**

1. アクセシビリティ許可
2. 対象アプリが前面か確認
3. アプリ側ショートカットが変わっていないか確認（クイックリファレンス参照）

**プロファイルが出ない**

`install.sh` を再実行し、Ulanzi Studio を完全終了→再起動。

## 9. アンインストール

```bash
launchctl unload ~/Library/LaunchAgents/com.vibe.deck.bridge.plist 2>/dev/null || true
rm -f ~/Library/LaunchAgents/com.vibe.deck.bridge.plist
rm -rf ~/Library/Application\ Support/Ulanzi/UlanziDeck/Plugins/com.vibe.deck.status.ulanziPlugin
pkill -f vibe-deck/bridge/dist/index.js || true
```

プリセットは Ulanzi Studio のプロファイル一覧から削除できます。
