# Vibe Deck クイックリファレンス

## 色＋ラベル（English only）

| 色 | ラベル | 状態 |
|----|--------|------|
| 白 | Idle | idle |
| 青 | Thinking | thinking |
| 緑 | Done | done |
| 橙 | Input | needs_input |
| 赤 | Error | error |
| 灰 | Ready | unused lane（押すとアプリ前面化） |

## 本体ナビ

| 操作 | 機能 |
|------|------|
| 下段 左ボタン | 前のプロファイル（全 N 個を巡回） |
| 下段 右ボタン | 次のプロファイル |
| 右端ダイヤル | ページ 1 / 2 / 3 |

## Page 1 Control

上段 A1–A5 = ライブ色  
Accept / Reject / New / Stop / Model など（ツール別テーマ）

## Page 2 Skills

共通: Design · Implement · Review · Fix · Test · Explain · Commit · Summary

| ツール | 追加 |
|--------|------|
| Cursor | Composer · Agent · Terminal |
| Codex | New Chat · Dictation · Model |
| Claude Code | New · Diff · Stop |

Skills キーはプロンプトをクリップボードへ入れ、⌘V で貼り付けます。

## Page 3 More

Settings · Help · Refresh

## デザイン

- Codex = 黒＋ティール
- Cursor = ダーク＋青
- Claude Code = 紫＋オレンジ

## ブリッジ

```bash
curl http://127.0.0.1:17823/health
curl 'http://127.0.0.1:17823/status?tool=cursor'
node scripts/wire-deck.mjs   # ナビ再配線
```

| | Codex | Claude Code | Cursor |

| Stop | ⌘. | Esc | ⌘⇧⌫ |
| Accept | ⌘Enter | ⌘Enter | ⌘Enter |
